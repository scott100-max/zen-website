Step 1: Generate Voice Audio

  1. Go to elevenlabs.io and sign up
  2. Choose a voice — recommended: calm, warm, British or neutral accent. Try "Rachel", "Adam", or browse the voice library
  3. Open a .txt script file from content/scripts/
  4. Paste the full text into ElevenLabs
  5. Set stability to ~70% and clarity to ~80% for a natural, warm tone
  6. Generate and download the MP3

Step 2: Download Background Audio

  1. Go to pixabay.com/music
  2. Search for the right ambience for each video:

Morning Meditation"ambient morning", "gentle birds", "calm piano"

Breathing for Anxiety"calm ambient", "soft pad", "meditation background"
Science of Mindfulness"minimal background music", "documentary ambient"
  1. Download the MP3 — no attribution needed from Pixabay

Step 3: Combine in Audacity

  1. Download Audacity (free, Windows/Mac)
  2. Import the voice MP3 (File → Import → Audio)
  3. Import the ambient MP3 on a second track
  4. Lower the ambient track volume to about -18 dB (use the track volume slider)
  5. Trim or loop the ambient to match the voice length
  6. File → Export → Export as MP3
  7. Save to content/audio/ using the naming convention: 01-morning-meditation.mp3
Once you save the MP3 to content/audio/, the audio players on the media page will work automatically — no code changes needed.

Step 4: Create YouTube Video

  1. Open Canva (free account works)
  2. Create a new design: YouTube Video (1920x1080)
  3. Set a background matching the thumbnail colour for that video
  4. Add the Zenscape logo and title text
  5. Upload your combined MP3 audio
  6. Set the page duration to match the audio length
  7. Download as MP4
  8. Upload to YouTube with title, description, and tags (see below)

Step 5: Create Thumbnail

  1. In Canva, create: YouTube Thumbnail (1280x720)
  2. Reference content/thumbnails/thumbnails.html for the design of each
  3. Use bold Inter font, brand colours, keep it clean
  4. Upload as the custom thumbnail when publishing on YouTube

Step 6: Update the Website

Once you've uploaded to YouTube:

  1. Get the YouTube video ID from the URL (the part after v=)
  2. Open media.html
  3. Find the HTML comment for that video's YouTube embed
  4. Uncomment it and replace YOUR_ID with the actual video ID
  5. Update the "Subscribe on YouTube" button href with your channel URL

Content Pipeline — Producing More

When you want new content, ask Claude to:

  • Write a new meditation script on any topic
  • Generate the YouTube title, description, and tags
  • Update the media page with a new card

Then run the same 6-step workflow above. Each new piece of content should take roughly 30–45 minutes of your time.
