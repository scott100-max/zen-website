Production Workflow

3 steps. ~10 minutes per video. Claude does everything else.

What Claude does: Writes scripts, YouTube titles, descriptions, tags, Canva prompts, website updates.
What you do: Paste, click, upload. That's it.

Step 1 — Generate Voice ~2 min

  1. Go to elevenlabs.io
  2. Open the .txt script file from content/scripts/
  3. Paste the full text in
  4. Pick a calm voice (use the same one every time for brand consistency)
  5. Click Generate. Download the MP3.

Step 2 — Create Video ~5 min

  1. Go to canva.com
  2. Create new: YouTube Video (1920x1080)
  3. Open the upload file from content/youtube-uploads/ for this video
  4. Paste the Canva AI prompt into Magic Media to generate a background visual
  5. Upload your ElevenLabs MP3 as the audio
  6. Set page duration to match audio length
  7. Export as MP4

Step 3 — Upload to YouTube ~3 min

  1. Go to YouTube Studio
  2. Upload your MP4
  3. Open the same upload file from content/youtube-uploads/
  4. Copy-paste the title, description, and tags
  5. Create a thumbnail in Canva (1280x720) using the thumbnail notes
  6. Publish
After uploading: Give Claude the YouTube video IDs and he'll update the website automatically.

File Locations

content/scripts/*.txt — Paste into ElevenLabs

content/youtube-uploads/*.txt — Everything for each video (Canva prompt + YouTube title/desc/tags)

content/thumbnails/thumbnails.html — Visual thumbnail references

Adding New Content

Ask Claude to write a new video. He'll create the script, upload package, and update the website. You just run the 3 steps above.
